# Challengue-TelecomX-LATAM

Este proyecto analiza por qué los clientes de una empresa de telefonía deciden cancelar su servicio (Churn). 

## Lo que hice en este proyecto:
 **Carga de Datos:** Convertí un archivo JSON con más de 7,000 clientes en una tabla fácil de leer.
 
**Limpieza:** Arreglé los nombres de las columnas, corregí errores en los precios y eliminé datos vacíos.

 **Análisis:** Creé gráficos para entender el perfil de los clientes.
 

## Conclusión Principal
Los clientes que se van son aquellos que pagan las facturas más altas (promedio de **$74 USD**), mientras que los que se quedan pagan menos (**$61 USD**). La empresa debe revisar sus precios para no perder más clientes.

##  Herramientas
* Python
* Pandas (para la tabla)
* Seaborn (para los gráficos)
